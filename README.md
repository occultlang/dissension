# Dissension
A privacy focused chatting protocol for everyone.

# Core of the protocol 
Reputation based moderation <br>
P2P <br>
Self copying servers on PC, and a mirror on mobile <br>
E2EE + Quantum Resistant <br>
Phone number for registration <br> 
On device AI model to prevent illegal content, and also maybe a jury-based governance for moderation per user and per server. <br>
